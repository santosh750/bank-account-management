# 🏦 Bank Account Management System

A full-stack **Bank Account Management System** built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This application helps manage bank accounts and departments with a clean dashboard interface.

## 🚀 Live Demo

Frontend Deployment (Vercel):

🔗 [https://your-vercel-url.vercel.app](https://bank-account-management-one.vercel.app/)

---

## 📌 Features

### Dashboard
- Modern bank management dashboard
- Overview of accounts and departments
- Sidebar navigation

### Account Management
- Add new bank accounts
- View all accounts
- Edit account details
- Delete accounts
- Manage account balance information

### Department Management
- Add departments
- View department list
- Store department details in MongoDB

### Backend API
- REST API using Express.js
- MongoDB database integration
- CRUD operations for accounts and departments
